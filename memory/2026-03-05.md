## 📋 笔记发布经验

**问题与挑战**：
1. PowerShell 解析复杂 JSON 字符串时遇到编码问题
2. MCP 服务需要正确的会话初始化流程
3. Accept 头需要包含 `application/json` 和 `text/event-stream` 两个值

**解决方案**：
- 先调用 `initialize` 方法初始化会话
- 再发送 `notifications/initialized` 通知
- 最后调用 `tools/call` 发布内容
- Python 脚本方式更稳定

## 💡 注意事项

1. **图片 URL 稳定性**: 使用静态 URL 作为图片源，确保链接长期有效
2. **标题长度**: 小红书要求标题不超过 20 个字
3. **正文长度**: 正文不能超过 1000 个字
4. **账号登录**: 小红书账号不允许在多个网页端同时登录
5. **风控风险**: 避免使用违禁词，防止被限流
