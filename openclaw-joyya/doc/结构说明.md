## 📂 backend/ - Spring Boot 后端项目

### 项目结构

```
backend/
├── pom.xml                              # Maven 项目配置文件
└── src/main/
    ├── java/com/joyya/                 # Java 源代码目录
    │   ├── JoyyaApplication.java       # 主应用类（启动入口）
    │   ├── config/                     # 配置类目录
    │   │   ├── SecurityConfig.java     # Spring Security 安全配置
    │   │   └── GlobalExceptionHandler.java # 全局异常处理器
    │   ├── controller/                 # 控制器目录
    │   │   └── AuthController.java     # 认证控制器（登录/注册）
    │   ├── dto/                        # 数据传输对象目录
    │   │   └── LoginRequest.java       # 登录请求 DTO
    │   ├── entity/                     # 实体类目录
    │   │   └── User.java               # 用户实体类
    │   ├── repository/                 # 数据访问层目录
    │   │   └── UserRepository.java     # 用户数据访问接口
    │   └── service/                    # 业务逻辑层目录
    │       └── UserService.java        # 用户服务类
    └── resources/                      # 资源配置文件目录
        ├── application.yml             # Spring Boot 应用配置
        └── schema.sql                  # 数据库初始化 SQL 脚本
```

### 目录详细说明

| 路径 | 说明 | 主要文件 |
|-----|------|--------|
| `backend/` | 后端项目根目录 | pom.xml |
| `backend/pom.xml` | Maven 依赖配置 | 所有后端依赖 |
| `backend/src/main/java/com/joyya/` | Java 源代码根目录 | JoyyaApplication.java |
| `backend/src/main/java/com/joyya/config/` | 配置类目录 | SecurityConfig.java, GlobalExceptionHandler.java |
| `backend/src/main/java/com/joyya/controller/` | 控制器层 | AuthController.java |
| `backend/src/main/java/com/joyya/dto/` | 数据传输对象 | LoginRequest.java |
| `backend/src/main/java/com/joyya/entity/` | 实体类 | User.java |
| `backend/src/main/java/com/joyya/repository/` | 数据访问层 | UserRepository.java |
| `backend/src/main/java/com/joyya/service/` | 业务逻辑层 | UserService.java |
| `backend/src/main/resources/` | 资源文件目录 | application.yml, schema.sql |
| `backend/src/main/resources/application.yml` | 应用配置文件 | 数据库、端口、JWT 配置 |
| `backend/src/main/resources/schema.sql` | 数据库初始化脚本 | users 表创建脚本 |

## 🔧 核心文件说明

### 后端核心文件

#### 1. `backend/pom.xml`
**用途：** Maven 项目配置文件
**内容：**
- Spring Boot 3.x 版本
- Spring Security, JPA, Web 依赖
- MySQL, H2 数据库驱动
- JWT 认证库
- Lombok 注解库

#### 2. `backend/src/main/java/com/joyya/JoyyaApplication.java`
**用途：** Spring Boot 启动入口
**功能：**
- @SpringBootApplication 注解
- main 方法启动应用

#### 3. `backend/src/main/java/com/joyya/config/SecurityConfig.java`
**用途：** Spring Security 安全配置
**功能：**
- CSRF 禁用（JWT 认证不需要）
- 认证接口允许匿名访问
- 禁用表单登录
- BCrypt 密码编码器配置

#### 4. `backend/src/main/java/com/joyya/config/GlobalExceptionHandler.java`
**用途：** 全局异常处理器
**功能：**
- 捕获所有未处理异常
- 返回统一错误格式
- Problem Details API 标准

#### 5. `backend/src/main/java/com/joyya/controller/AuthController.java`
**用途：** 认证控制器
**功能：**
- POST /api/auth/login - 用户登录
- POST /api/auth/register - 用户注册
- 返回统一响应格式

#### 6. `backend/src/main/java/com/joyya/entity/User.java`
**用途：** 用户实体类
**功能：**
- @Entity, @Table 注解
- id, username, email, password 字段
- createdAt, updatedAt 时间戳
- @PrePersist, @PreUpdate 回调

#### 7. `backend/src/main/java/com/joyya/service/UserService.java`
**用途：** 用户服务类
**功能：**
- 用户注册
- 用户登录认证
- 用户名查询
- BCrypt 密码加密/验证

#### 8. `backend/src/main/java/com/joyya/repository/UserRepository.java`
**用途：** 数据访问层
**功能：**
- JpaRepository 接口
- findByUsername, findByEmail 查询
- existsByUsername, existsByEmail 检查

#### 9. `backend/src/main/resources/application.yml`
**用途：** 应用配置文件
**内容：**
- Spring 配置（数据源、JPA）
- MySQL/H2数据库配置
- JWT 密钥和过期时间
- 服务器端口配置

#### 10. `backend/src/main/resources/schema.sql`
**用途：** 数据库初始化脚本
**功能：**
- 创建 users 表
- 添加索引
- 插入测试数据

## 📊 项目架构

### 技术栈概览

```
前端层：
├── Vue 3 - 渐进式 JavaScript 框架
├── Vite - 下一代前端构建工具
├── Element Plus - Vue 3 UI 组件库
├── Pinia - Vue 状态管理
└── Vue Router - 路由管理

后端层：
├── Spring Boot 3.x - Java 应用框架
├── Spring Security - 安全框架
├── Spring Data JPA - ORM 框架
├── BCrypt - 密码加密
└── JWT - 令牌认证

数据层：
├── MySQL 8.0 - 关系数据库
└── H2 - 内存数据库（开发/测试）
```

### 目录对应关系

| 层级 | 目录 | 说明 |
|-----|------|------|
| **前端视图层** | `frontend/src/views/` | 页面组件 |
| **前端 UI 层** | `frontend/src/components/` | 公共组件 |
| **前端资源层** | `frontend/src/assets/` | 静态资源 |
| **前端路由层** | `frontend/src/router/` | 路由配置 |
| **前端配置层** | `frontend/src/` | 入口配置 |
| **后端控制器层** | `backend/.../controller/` | HTTP 请求处理 |
| **后端服务层** | `backend/.../service/` | 业务逻辑 |
| **后端数据层** | `backend/.../repository/` | 数据访问 |
| **后端实体层** | `backend/.../entity/` | 数据库映射 |
| **后端配置层** | `backend/.../config/` | 系统配置 |

## 📝 使用指南

### 启动后端
```bash
cd backend
mvn clean install
mvn spring-boot:run
```
访问：http://localhost:8080

### 启动前端
```bash
cd frontend
npm install
npm run dev
```
访问：http://localhost:5173

### 数据库配置
- 使用 MySQL：修改 application.yml 配置 MySQL 连接
- 使用 H2：使用默认配置，无需安装
