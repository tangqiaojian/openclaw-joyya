## 如何更新此日志

每次完成新的功能或修复 Bug 时，请在此处添加相应描述。

**格式说明：**
- **未发布**：当前正在开发中的功能
- **[版本号]**：已发布的版本，格式为 `v主版本。次版本.修订版本`
- **分类**：添加、修改、修复
- **日期**：YYYY-MM-DD 格式

**示例：**

```
## [v1.0.2] - 2026-03-04

### 添加

- 新增用户资料编辑功能
- 支持头像上传和修改

### 修改

- 优化了登录页面加载速度
- 改进了 API 响应时间

### 修复

- 修复了密码验证逻辑 Bug
- 修复了表单提交错误提示
```
